Git Basic


Basic Command Flow

1. git init : initialize git storage
2. git status : check current git storage status
3. git branch : make new branch of my own
4. git commit : most important command, i.e. git commit -m "Message here"
5. git push : any modification made in my local computer can be seen in online too
6. git pull : re-install the git storage into latest version 7. git merge : my branch will merge into master branch where all co-workers can view

1. git add .
2. git commit -m "Message"
3. git push

To start with my local computer:
1. type command: git config --global user.name "my name here"
2. type email: git config --global user.email "my email here"
3. Get space for my respiratory in my local directory, i.e. mkdir ~/MyProject
4. cd ~/MyProject and use git command: git init
5. touch ReadMe.txt will create a ReadMe.txt in MyProject folder
6. git add ReadMe.txt
7. git commit -m "Add ReadMe.txt"
8. Before to 'push' the local work into online storage, we need to connect them first
9. git remote add origin https://github.com/username/myproject.git (-> I can find URL from respiratory)
10. And now I can upload changes made by git push
