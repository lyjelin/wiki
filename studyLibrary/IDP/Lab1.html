Task 3

Develop application ultrasonic_buzzer.ino so that the buzzer starts to beep when the object is 10cm away from the sensor. The beeping frequency keeps increasing and becomes long beep when the object is closer than 4cm. The beeping frequency is reduced if the object moves away. You can use either version of measure_distance() or develop your own one.
