I have created an external css file written as 'style.css' and add a < link > to html files.

But the problem was: once I link a css in to html file, the browser (I was using Safari and Chrome) didn't immediately apply the change. For example, when I add and modified the margin attribute in style.css, the web didn't looks like it received the modification. So I googled to find solution.

It was CSS file linkage error. As the browser saved the previous CSS cache, it was not updating to latest style.css version. And there were 2 solutions: first, removing all cookie&cache history in internet (deleting browser's cache) or second, add ?after to href link.

For example, if this is our code for linking external css file: < link rel="stylesheet" href="../style.css" &g update the href link part to href="../style.css?after" so the browser can be alert on every css file update.
